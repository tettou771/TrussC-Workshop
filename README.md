# TrussC Workshop

TrussC 워크숍용 샘플 코드입니다..

테마별로 프로젝트를 구성하였고, 프로젝트에는 각 테마의 사용법과 소스코드 후반부에 "챌린지" 코너가 있습니다.

먼저 실행해보고 (VSCode 등에서는 F5 키), 어떤 건지 확인한 다음 tcApp.h, tcApp.cpp를 살펴보세요. 아하~ 하고 이해가 됐으면 챌린지에 도전해보시기 바랍니다.

챌린지에는 정답이 없으니, 의도한대로 동작함다면 OK입니다.

## 샘플 종류

### 00-basicGraphics

기본적인 도형 그리기입니다. 선을 긋거나 다각형으로 영역을 만들 수 있습니다.

### 01-drawShape_vs_drawStroke

Processing에서는 두꺼운 stroke에 cap을 넣어서 끝을 둥글게 하거나, 꺾인 선의 모서리를 깎을 수 있습니다. openFrameworks에서는 이게 안 돼서 불만이었는데, ofxStroke라는 애드온을 만들어 사용해야 했습니다.

TrussC에서는 이러한 기능도 내장되어 있어서 그래픽 표현이 좀 더 쉬울겁니다.

### 02-tween

애니메이션할 때 부드럽게 가감속하는 예제입니다.

그냥 사각형이라도 두부처럼 보이거나, UI의 기분 좋은 느낌을 움직임으로 표현할 수 있어서 꽤 중요한 요소입니다. 근데 tween은 외부 애드온 없이는 쓰기 어려운 경우가 많아서, TrussC에서는 내장하였습니다.

### 03-mouse_key

마우스와 키 이벤트 처리 방법입니다.

간단하기 떄문에 가볍게 훑어보면 됩니다만, 키 입력은 대문자로 처리한다는 점만 주의해주세요. 예를 들어 `if (key == 'C')` 이런 식입니다.

### 04-transform

좌표를 이동하거나 회전하거나 스케일링하는 예제입니다.

`pushMatrix()`, `popMatrix()` 같은 메서드는 oF나 Processing에도 있으니까 익숙할 수도 있습니다.

### 05-mathGenerative

sin, cos, noise를 활용한 제너러티브 그래픽스 예제입니다.

파형(wave)를 그리거나, 리사주(Lissajou's Curve) 곡선을 그리거나, noise로 유기적인 움직임을 만들수 있습니다. 크리에이티브 코딩의 묘미는 바로 이것이라고 할수 있죠.

### 06-imageTexture

픽셀 단위로 이미지를 조작하는 예제입니다.

Image를 allocate하고, setColor로 픽셀을 쓰고, update로 GPU에 업로드하고, draw로 표시하는 흐름입니다. 이 과정을 이해하면 절차적인(Procedural) 텍스처 생성이나 실시간 이미지 처리를 할 수 있습니다.

### 07-nodeSystem

RectNode를 사용한 부모-자식 관계를 다루는 예제입니다.

addChild로 부모에 자식을 추가하면 부모가 움직일 때 자식도 따라옵니다. 로컬 좌표로 그려지니까 UI 패널이나 버튼을 만들 때 편리하죠. enableEvents()를 호출하면 마우스 호버나 클릭도 감지할 수 있습니다.

### 08-fbo

FBO(오프스크린 렌더링) 예제입니다.

begin() / end() 사이에 그린 것이 텍스처가 됩니다. TextureFormat 지정 방식은 oF의 그것과는 다른 enum이니 주의해주세요.

### 09-sound

beep()과 ChipSound 예제입니다.

beep()은 내장 프리셋 사운드로, 디버그나 UI 피드백에 쓸 수 있습니다. ChipSound는 파형, 주파수, 길이를 지정해서 직접 사운드를 만드는 구조입니다. 패미컴 느낌의 효과음을 간단하게 만들 수 있습니다.
